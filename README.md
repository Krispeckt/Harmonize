## Features
<h4>

- Simple and easy to use
- Multi-node support
- Last frequency used cache for queries
- Audio filters
- Full asynchronous design

</h4>

---

## Library installing

<h4>

Development version


```shell
pip install git+https://github.com/Krispeckt/Harmonize.git
```

Default

```shell
pip install harmonize.dev
```

</h4>
